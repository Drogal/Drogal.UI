# Drogal UI

Drogal UI is a user interface library components for React from Drogal, Rede de Farm√°cias . See [Drogal](https://www.drogal.com.br/) HomePage.

## Installation

Use the package manager NPM to install.

```bash
npm install drogal-ui
```

Use the package manager YARN to install.

```bash
yarn add drogal-ui
```

## Theming

import the css file into your main javascript page

```bash
import 'drogal-ui/dist/index.css'
```

## Usage

```python
import { Button } from 'drogal-ui'

export default function MyComponent() {
  return (
    <Button label="Drogal UI" />
  )
}
```
